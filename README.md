# Gravity Forms Checkout.com Payment Gateway Pro

Professional Checkout.com payment gateway for Gravity Forms with Frame and Component support.

## Features

- **Dual Payment Methods**: Checkout Frame and Checkout Component
- **3D Secure Authentication**: Enhanced security
- **Webhook Integration**: Real-time payment updates
- **Professional Code**: WordPress coding standards
- **Flexible Settings**: Global defaults with per-form overrides

## Installation

1. Upload plugin to `/wp-content/plugins/gravityforms-checkout-com-pro/`
2. Activate the plugin
3. Configure API keys in Forms > Settings > Checkout.com Pro
4. Create form feeds with Checkout.com Pro payment method

## Requirements

- WordPress 5.0+
- PHP 7.4+
- Gravity Forms 2.4.0+
- Checkout.com merchant account

## Development

This plugin follows WordPress coding standards and uses a modular architecture for maintainability.
